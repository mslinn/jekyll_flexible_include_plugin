Renders as:

{% flexible_include pre highlight='RakeTask' '../Rakefile' %}

Renders as:

{% flexible_include pre file='../Rakefile' %}

Inside Include with copyButton and number Keyword Options

{% raw %}{% flexible_include copyButton number file='_bin/debug' %}{% endraw %}

{% flexible_include copyButton number file='_bin/debug' %}
